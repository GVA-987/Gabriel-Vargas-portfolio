# Portafolio: Juan Gabriel Vargas Aliaga (Full-Stack & IoT Engineer)

Este repositorio contiene el código fuente de mi portafolio profesional en línea. Está diseñado para presentar mi experiencia como **Desarrollador Full-Stack** con una especialización distintiva en **Ingeniería de Soluciones IoT (Internet de las Cosas)**.

El objetivo del diseño es simular una interfaz de consola oscura con un estilo neón (Dark/Neon Aesthetic) para destacar mis habilidades técnicas y proyectos.

## Stack Tecnológico

- **Frontend Framework:** React.js
- **Lenguaje:** JavaScript (ES6+)
- **Estilos:** SASS/SCSS (Con una arquitectura modular BEM y Mixins)
- **Control de Versiones:** Git & GitHub
